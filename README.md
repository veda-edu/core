# 🔥 VEDA — Next-Gen LMS Platform

> **VEDA** — це сучасна платформа для студентів, яка об'єднує навчання, дедлайни, розклад і успішність в одному стильному dashboard.

---

## ✨ Про проєкт
  
Це **student-first платформа**, яка робить навчальний процес зручним, зрозумілим і приємним.

📚 Всі дисципліни  
⏰ Всі дедлайни  
📊 Вся аналітика  
     — в одному місці

---

## 🚀 Основні можливості

- 📌 **Dashboard студента**
  - швидкий доступ до всього важливого

- 🔥 **Система дедлайнів**
  - пріоритетні завдання
  - підсвітка термінових ("дедлайнів")

- 📊 **Статистика та аналітика**
  - GPA
  - відвідуваність
  - активні завдання

- 📚 **Курси**
  - список дисциплін
  - прогрес по кожному курсу

- 🗓 **Розклад**
  - таймлайн занять
  - активна пара в реальному часі

- 💬 **Фідбек та оцінки**
  - коментарі викладачів
  - результати робіт
